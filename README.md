# U-Net Reproduction and Improvements

A reproducible implementation of **U-Net** for biomedical image segmentation,
including baseline reproduction, experimental analysis, and controlled
improvement studies.

## Goals

- Reproduce the original U-Net architecture and training procedure.
- Build a clear and repeatable data preprocessing and evaluation pipeline.
- Compare proposed improvements with the baseline through ablation experiments.
- Record configurations, results, and conclusions for each experiment.

## Project status

Environment setup and experiment scaffolding are in progress.

## Reference

Ronneberger, O., Fischer, P., & Brox, T. (2015).  
*U-Net: Convolutional Networks for Biomedical Image Segmentation*.
